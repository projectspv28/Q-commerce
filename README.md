# WHISTLE

WHISTLE is a quick-commerce platform built with Next.js 16 (App Router), delivering groceries and daily essentials in minutes with a bold yellow and red brand identity.

## Run locally
1. Install deps: `npm install`
2. Copy `.env.local` and fill required values (MongoDB, NextAuth secret, Google OAuth, Cloudinary, Stripe keys, socket server URL, email credentials).
3. Start dev server: `npm run dev` then open `http://localhost:3000`.

## Available scripts
- `npm run dev` - start development server
- `npm run build` - production build
- `npm run start` - run compiled app
- `npm run lint` - lint with ESLint

## Tech stack
Next.js, React 19, TypeScript, Tailwind (v4), NextAuth, MongoDB/Mongoose, Stripe, Cloudinary, Socket.IO.
